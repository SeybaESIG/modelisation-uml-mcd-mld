# **Modélisation UML, MCD et MLD**

## **Description**

Ce dépôt regroupe un ensemble de diagrammes UML et de modèles de données (MCD, MLD) réalisés dans le cadre de différents cas d’étude.

## **Types de diagrammes présents**

* Diagrammes de cas d’utilisation (Use Case)
* Diagrammes d’activités (Activity)
* Diagrammes de séquence (Sequence)
* Diagrammes de communication (Communication)
* Diagrammes de classes (Class)
* Modèles conceptuels de données (MCD)
* Modèles logiques de données (MLD)
* Diagrammes entité-relation (ERD)
